# GTFS Honduras Open - Transporte Público Abierto

**Proyecto open-source de datos GTFS para el transporte público en Honduras**

Este repositorio contiene un dataset GTFS (General Transit Feed Specification) para autobuses interurbanos y ferries en Honduras.

### Incluye actualmente:
- Rutas de ferries a las Islas de la Bahía (La Ceiba ↔ Roatán, La Ceiba ↔ Utila, Utila ↔ Roatán)
- Rutas principales de **Transportes Cristina**, **Mirna Express** y **Hedman Alas**
- Terminales importantes y muelles de ferries con coordenadas reales
- Estructura básica de una app Flutter para visualizar las paradas en mapa

### Objetivo
Crear el primer dataset GTFS nacional abierto de Honduras para que pueda ser usado por:
- Google Maps
- Otras apps de transporte
- Desarrolladores
- Autoridades de transporte

### Archivos GTFS incluidos
- agency.csv
- stops.csv
- routes.csv
- trips.csv
- stop_times.csv
- calendar.csv

### Cómo contribuir
1. Haz un Fork del repositorio
2. Mejora los horarios, agrega nuevas rutas o corrige coordenadas
3. Envía un Pull Request con tus cambios

Cualquier persona puede ayudar a mantener los datos actualizados (horarios cambian frecuentemente).

**English**

# GTFS Honduras Open - Open Public Transit Data

**Open-source GTFS dataset for public transportation in Honduras**

This repository contains GTFS data for intercity buses and ferries in Honduras.

### Currently includes:
- Bay Islands ferry routes (La Ceiba ↔ Roatán, La Ceiba ↔ Utila)
- Main routes from Transportes Cristina, Mirna Express, and Hedman Alas
- Major bus terminals and ferry docks with accurate coordinates
- Basic Flutter app skeleton (map + trip planner)

### Purpose
To create the first national open GTFS feed for Honduras so it can be used by Google Maps, other transit apps, developers, and authorities.

### How to contribute
Feel free to add more routes, improve schedules, or fix stop locations. Pull requests are welcome!
