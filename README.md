# GTFS Honduras Open Source

Este proyecto contiene un feed de datos GTFS (General Transit Feed Specification) de código abierto para el transporte público en Honduras (operadores interurbanos y ferris de las Islas de la Bahía).

This project contains an open-source GTFS (General Transit Feed Specification) data feed for public transportation in Honduras (intercity operators and Bay Islands ferries).

## Contenido (Contents)

- `agency.csv`: Operadores de transporte (Bay Islands Ferries, Transportes Cristina, Mirna Express, Hedman Alas, TICA BUS).
- `stops.csv`: Terminales y puertos (con coordenadas).
- `routes.csv`: Rutas cubiertas por los operadores.
- `trips.csv`: Viajes específicos asignados a las rutas.
- `stop_times.csv`: Horarios de salida y llegada para cada viaje.
- `calendar.csv`: Días de servicio.

## Cómo contribuir (How to contribute)
Para contribuir o reportar cambios en rutas y horarios, por favor modifica los archivos CSV en tu fork del repositorio y abre un Pull Request (PR) o crea un Issue detallado.
